## Quick Start

<div class="steps-container">
  <div class="step-card">
    <div class="step-number">1</div>
    <div class="step-content">
      <h3>Clone and Install</h3>

```bash
git clone https://github.com/LeagueToolkit/lol-meta-wiki.git
cd lol-meta-wiki
pnpm install && bun install
```

  </div>
  </div>

  <div class="step-card">
    <div class="step-number">2</div>
    <div class="step-content">
      <h3>Generate Database</h3>

```bash
pnpm generate-db
```

  </div>
  </div>

  <div class="step-card">
    <div class="step-number">3</div>
    <div class="step-content">
      <h3>Start Development Server</h3>

```bash
pnpm dev
```

  <p class="step-note">Visit <code>http://localhost:4321</code> to see the wiki</p>
  </div>
  </div>
</div>

## Contribute Documentation

<CardGrid>
  <Card title="Via Web" icon="external">
    Click "Add documentation" buttons on any class page, edit on GitHub, and submit a PR. No local setup required!
  </Card>
  <Card title="Locally" icon="pencil">
    Create or edit YAML files in `db/docs/` with full Markdown support for rich documentation.
  </Card>
</CardGrid>
